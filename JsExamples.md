## CRUD Operations:
###### 1. ARRAYS:

```js
    let arr = new Array ();
    let arr = [];
    let arr = [‘1’,’A’,’a’];
```
  + ###### ADDING DATA	
```js
          arr.push(‘x’);            //@end          [‘1’,’A’,’a’,’x’];
          arr[arr.length] = ‘x’;    //@end          [‘1’,’A’,’a’,’x’];
          arr.unshift(‘x’);         //@beginning    [‘x’,‘1’,’A’,’a’];
```
+ ###### DELETING
```js
          arr.pop();            //@end              [‘1’,’A’];
          arr.shift();          //@beginning        [’A’,’a’];
```
